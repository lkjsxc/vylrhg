# Glossary

- **Module ID**: A globally unique identifier for a markup/assembly bundle.
- **Tab Context**: The isolated VM + state snapshot attached to a tab.
- **Render Tree**: Immutable tree derived from markup and state.
- **Render Ops**: Deterministic low-level draw instructions.
- **Effect**: Explicit, typed request for side-effectful work.
- **Event Bus**: Message transport between subsystems.
- **Layout Tree**: Immutable tree encoding spatial relationships.
- **Sidecar Node**: Markup node that binds to assembly handlers.
