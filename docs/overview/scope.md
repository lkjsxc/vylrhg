# Scope

- Product: Super App combining tabbed UI, tiling, and browser-like rendering.
- Platform: Rust application with async runtime using tokio.
- UI: terminal/desktop hybrid abstraction with Vim-like control flow.
- Languages: custom `vylrhg-assembly` and `vaylrhg-markup` with shared modules.
- Execution: local sandboxed evaluation, no remote execution assumed.
- Non-goals: backwards compatibility, legacy browser compatibility.
